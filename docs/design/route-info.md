## 4. 경유 정류장 및 경로

### 4-1. 장유 노선

```
[장유 방면] 갑을병원 → 코아상가 → 장유농협 → 장유IC → (남해고속도로) → 사상IC → 사상터미널
[사상 방면] 사상터미널 → 사상IC → (남해고속도로) → 장유IC → 장유농협 → 코아상가 → 갑을병원
             └─ 추가 하차: 장유1동 행정복지센터
```

### 4-2. 율하 노선

```
[외고 방면] 사상터미널 → 사상IC → 서부산낙동강교 → 가락IC → 해포 → 봉림교차로 → (장유로/조만로)
            → 율하2지구입구 → E편한 아파트 → 김해외고

[사상 방면] 김해외고 → 율하2지구입구 → (조만로/장유로) → 봉림교차로 → 해포 → 가락IC
            → 서부산낙동강교 → 사상IC → 사상터미널
```

> **※ 남장유IC는 부산 방면 진입 램프 없음 → 가락IC를 통해 고속도로 진입**

> **주의 사항**
> - **해포 정류장**: 정차 안내 방송이 나오나, 수요 없으면 무정차 통과 가능 → 탑승/하차 시 기사님께 출발 전 사전 고지 필요. 기본 요금 적용
> - **율하 파출소 정류장**: 안내 방송 명칭과 실제 정류장 명칭이 다름 (2026년 1월부터 이 정류장 폐지, 율하중심상가·중앙하이츠로 대체)
> - **가락IC 구조 주의**: 톨게이트 통과 직후 **차단봉으로 인해 차로 변경 불가** → 톨게이트 진입 전에 차로 선택 완료 필요. 창원행 차량이 사상 방향으로 착오 진입하는 사례 빈번
> - **사상 방향 차로 변경**: 서부산낙동강교 구간이 점선 차선이므로 방향 지시등 켜고 차로 변경 가능

> **2026년 1월부터 변경된 경유지**
> ```
> [구] 율하파출소 경유 (정류장 3곳)
> [신] 율하중심상가 → 중앙하이츠 아파트 경유 (정류장 4곳)
> ```

## 7. 주요 특이사항 및 운영 정책

### 7-1. 방학 감차
- 학생 통학 수요 높음 → **방학 기간 중 감차 운행**
- 방학 배차는 **현장 정류장에서 직접 확인** 필요

### 7-2. 일요일 감차
- 이 노선은 주말 관광보다 **평일 출퇴근·통학 수요** 중심
- 일요일 감차 폭이 크며 낮 시간대 최대 40분 공백 발생
- 40분 대기 대신 44번 버스 → 김해 시내 → 경전철 이용 권장

### 7-3. 율하 노선 경로 변경 (2026년 1월 1일)
- 율하 노선이 **율하중심상가, 중앙하이츠 아파트 경유**로 변경

### 7-4. 다인승 탑승 시 주의
- 3인 이상 탑승 시 교통카드 단말기 조작 시간 지연 가능
- 카드 대신 **승차권 사용 권장**

### 7-5. 선불 교통카드 잔액
- **5,000원 이상** 충전 권장 (미만 시 잔액 부족 경고)

### 7-6. 도로 인프라 주의 구간
- **가락IC**: 톨게이트 통과 후 차로 좁아짐, 창원 방향 착오 진입 빈번 → 진입 전 차로 확인 필수
- **사상IC 인근 / 서부산휴게소**: 통행량 많아 속도 저하 가능
