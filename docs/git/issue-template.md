# Issue Template

## 제목 형식

```
{Area}: {설명}
```

### Area 목록

| Area       | 설명                      |
| ---------- | ------------------------- |
| `Timetable`| 시간표 표시, 리스트       |
| `Schedule` | 평일/주말/공휴일 판단     |
| `NextBus`  | 다음 버스 찾기, 자동 스크롤 |
| `Data`     | JSON 데이터, 파싱         |
| `Cache`    | 캐싱, 오프라인 저장       |
| `Network`  | 네트워크, API 호출        |
| `Offline`  | 오프라인 모드             |
| `Notice`   | 공지 배너                 |
| `UI`       | UI/UX, 디자인             |
| `Settings` | 설정, 앱 정보             |
| `Feedback` | 제보하기, 피드백          |
| `Test`     | 테스트 코드               |
| `Build`    | 빌드, Xcode 설정          |
| `Release`  | 릴리즈, App Store         |
| `Docs`     | 문서                      |

### 예시

```
Timetable: 시간표 리스트 UI 구현
Schedule: 공휴일 주말 시간표 적용 버그 수정
NextBus: 다음 버스 자동 스크롤 기능 추가
Data: JSON 스키마 버전 호환성 처리
Offline: 오프라인 토스트 메시지 표시
Docs: README 업데이트
```

## 라벨 가이드

이슈 생성 시 적절한 라벨을 붙여주세요.

### 타입 라벨 (필수)

| 라벨            | 설명             |
| --------------- | ---------------- |
| `bug`           | 버그 수정        |
| `enhancement`   | 새 기능/개선     |
| `refactor`      | 리팩토링         |
| `documentation` | 문서 작업        |
| `chore`         | 설정, 빌드       |
| `test`          | 테스트 코드      |

### 우선순위 라벨 (선택)

| 라벨                 | 설명           |
| -------------------- | -------------- |
| `priority: critical` | 즉시 처리 필요 |
| `priority: high`     | 높은 우선순위  |
| `priority: medium`   | 중간 우선순위  |
| `priority: low`      | 낮은 우선순위  |

### 영역 라벨 (선택)

`area: timetable`, `area: schedule`, `area: data`, `area: network`, `area: ui` 등 해당 기능 영역
