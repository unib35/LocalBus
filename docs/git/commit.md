## 커밋 컨벤션

> 프로젝트에서는 다음과 같은 커밋 메시지 컨벤션을 사용합니다.
> 모든 커밋은 **한국어**, **50자 이내 제목**, **마침표 없이 작성**해야 합니다.

### 핵심 규칙

1. **한국어로 작성**: 모든 커밋 메시지는 한국어로 작성
2. **AI 도구 문구 금지**: "Generated by", "Co-Authored-By: Claude" 등 AI 관련 문구 포함 금지
3. **마침표 없음**: 제목 끝에 마침표 사용 금지
4. **50자 이내**: 제목은 간결하게

### 커밋 메시지 구조

```
[Type]: 커밋 제목 (한국어, 50자 이내)

- 세부 내용 1
- 세부 내용 2

Close #이슈번호
```

- **제목(Subject)**: 50자 이내, 한국어로 작성
- **본문(Body)**: 선택 사항이며, `-` 목록으로 작성 (한 줄당 72자 이내 권장)
- **이슈 번호**: 관련 이슈에 자동 연결되도록 `Close #번호` 형식 사용

### Type 목록

| Type         | 용도                                                  |
| ------------ | ----------------------------------------------------- |
| `[Add]`      | 기능과 무관한 코드 추가 (라이브러리, 유틸 함수 등)    |
| `[Chore]`    | 잡무성 작업 (버전 코드 수정, 파일 이동, 이름 변경 등) |
| `[Comment]`  | 주석 추가 및 변경                                     |
| `[Del]`      | 불필요한 코드/주석 삭제                               |
| `[Design]`   | UI 구현 및 뷰 관련 코드 수정                          |
| `[Docs]`     | 문서 작업 (README, Wiki 등)                           |
| `[Feat]`     | 새로운 기능 구현                                      |
| `[Fix]`      | 버그 수정 및 문제 해결                                |
| `[Merge]`    | 브랜치 병합                                           |
| `[Refactor]` | 코드 리팩토링 (로직은 유지하면서 구조 변경)           |
| `[Remove]`   | 파일 삭제                                             |
| `[Setting]`  | 프로젝트 세팅 관련 작업                               |
| `[Test]`     | 테스트 코드 추가                                      |
| `[Perf]`     | 성능 향상 관련 코드                                   |

### 기본 예시

[Feat]: 홈 화면 인기 플레이리스트 섹션 구현

- SectionView 컴포넌트 생성
- API 연동을 위한 ViewModel 추가

Close #17

### 잡무 예시

[Chore]: Tuist 버전 고정 및 generate 스크립트 추가

- tuist version 사용 고정
- bootstrap.sh에 generate 명령 포함

Close #6

### 버그 수정 예시

[Fix]: 이미지 캐싱 오류 해결

- Kingfisher placeholder 처리 추가
- 잘못된 URL 예외 처리

Close #21

---

### ⛔️ 안 되는 커밋 제목 예시

- `[Feat] 로그인 기능 구현.` ← ❌ 마침표 X
- `[Fix] Fixed the bug.` ← ❌ 영어 X (한국어로 작성)
- `커밋` ← ❌ Type 누락
- `[Refactor]:` ← ❌ 제목 없음
- `[Feat]: 기능 추가\n\nCo-Authored-By: Claude` ← ❌ AI 문구 X

### 🚀 기타

- 모든 커밋은 가능한 이슈와 연동해 `Close #이슈번호` 작성
- PR 단위로 의미 있는 단위로 커밋을 정리하고 **스쿼시 머지** 진행
- **AI 도구 사용 시에도 커밋 메시지에 AI 관련 문구 포함 금지**
