# LocalBus - Codex 프로젝트 가이드

## 프로젝트 개요

**LocalBus**는 장유-사상 시외버스 시간표를 제공하는 iOS 앱입니다.

### 핵심 목표
- **Zero-Click Info**: 앱 실행 즉시 다음 버스 시간 확인
- **Always Up-to-Date**: 앱 업데이트 없이 데이터 갱신으로 시간표 변경 반영
- **Context Aware**: 평일/주말/공휴일 자동 판단

### 기술 스택
- **Platform**: iOS 16.0+
- **Language**: Swift
- **UI Framework**: SwiftUI
- **Architecture**: MVVM
- **Testing**: XCTest, XCUITest
- **Data Source**: GitHub Raw JSON (원격) + Bundle JSON (로컬)

## Git 컨벤션 (필수 준수)

### 브랜치 네이밍
> 📖 상세: [docs/git/branch.md](./docs/git/branch.md)

```bash
{type}/{description}

# 예시
feat/next-bus-auto-scroll
fix/holiday-schedule-detection
refactor/timetable-service
```

### 커밋 메시지
> 📖 상세: [docs/git/commit.md](./docs/git/commit.md)

```bash
[Type]: 제목 (한국어, 50자 이내, 마침표 없이)

# Type 목록
[Feat]     # 새로운 기능
[Fix]      # 버그 수정
[Refactor] # 리팩토링 (기능 변경 없음)
[Design]   # UI 구현 및 뷰 관련
[Test]     # 테스트 코드
[Docs]     # 문서 작업
[Chore]    # 설정, 빌드, 잡무
[Setting]  # 프로젝트 세팅
```

**핵심 규칙:**
- ✅ **한국어**로 작성
- ❌ AI 관련 문구 금지 (Co-Authored-By, Generated by 등)
- ❌ 마침표 금지

**예시:**
```bash
[Feat]: 다음 버스 자동 스크롤 기능 구현
[Fix]: 공휴일 평일 시간표 표시 버그 수정
[Refactor]: DateService에서 시간 계산 로직 분리
```

### 마이크로 커밋 원칙
> 📖 상세: [docs/git/micro-strategy.md](./docs/git/micro-strategy.md)

- 하나의 논리적 변경 = 하나의 커밋
- 파일 1-3개 변경 시 즉시 커밋
- 테스트 통과 후 커밋
- 구조적 변경(리팩토링)과 행위적 변경(기능)은 별도 커밋

## 개발 가이드

### TDD 워크플로우
> 📖 상세: [docs/dev/kentback-tdd-guide.md](./docs/dev/kentback-tdd-guide.md)

```
Red → Green → Refactor

1. 실패하는 테스트 작성 (Red)
2. 최소한의 코드로 테스트 통과 (Green)
3. 리팩토링 (테스트 통과 상태 유지)
```

### 빌드 & 테스트

```bash
# 빌드
xcodebuild build -scheme LocalBusApp -destination 'platform=iOS Simulator,name=iPhone 15'

# 테스트
xcodebuild test -scheme LocalBusApp -destination 'platform=iOS Simulator,name=iPhone 15'
```
